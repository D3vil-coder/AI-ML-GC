## Sourcing Guidelines

1. **Resolution**: All images must be 1200x800 pixels
2. **Format**: JPG or PNG (JPG preferred for file size)
3. **Licensing**: Use royalty-free images from:
   - Unsplash (https://unsplash.com)
   - Pexels (https://pexels.com)
   - Pixabay (https://pixabay.com)
4. **No Branding**: Remove or avoid images with visible company logos
5. **Professional Quality**: High-resolution, well-lit, professional photography
